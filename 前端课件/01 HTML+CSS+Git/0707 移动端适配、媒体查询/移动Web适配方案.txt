整体上分为两种
一、伸缩式
  如：拉钩、Boss直聘
  技术：使用百分比，让宽度根据屏幕的宽度进行自适应，高度、文字大小不变


  流式布局：类似伸缩式，但是使用max-width控制页面的最大宽度

二、等比缩放式
  如：B站（vw），腾讯首页（rem）
  技术：
  1、使用rem，所有的尺寸和文字大小都是用rem设置
    1.1 使用JS控制html的文字大小，早期vw还没有普及的时候使用的方式，比如淘宝的flexible.js
    1.2 使用vw控制html字体的大小，当vw兼容主流的浏览器之后，推荐使用vw
  2、直接使用vw，所有的尺寸和文字大小都是用vw设置

  在等比缩放式，有些网站会有一个最大宽度使用max-width进行设置，如果使用rem同时约束html字体的最大值
