引入一下js
**以下的jquery和layer必须同时引入,否则没效果**
```js
<script src="https://cdn.bootcdn.net/ajax/libs/jquery/3.6.0/jquery.js"></script>
<script src="https://cdn.bootcdn.net/ajax/libs/layer/3.5.1/layer.min.js"></script>
```

使用
在js中加入
```js
layer.msg(弹出的内容)
//如下
layer.msg("你好")
```
